# マーケティング（マーケ） 向け スターターコンテキスト

> この役割の人がAI秘書を最速で戦力化するための初期セット。本体ガイド Step5（コンテキスト注入）から参照される。まずここから埋め、慣れたら広げる。
>
> 使い方：あなた（Claude Code）は、マーケ担当であるユーザーに以下を順にヒアリングし、`context/` を下書きしていく。先にブランドガイド・過去の記事/LP・ペルソナ定義で埋められる所を埋め、足りない分だけ聞く。

## 1. まず入れる context/ ファイル（最初に用意する自社情報）

- `context/company/overview.md` — 自社の提供価値・ターゲット・選ばれる理由（発信の軸になる）
- `context/company/brand.md` — トーン＆マナー・表記ルール・NGワード（生成物の声を揃える土台。逸脱を防ぐ）
- `context/marketing/personas.md` — ターゲット像／ペルソナ（誰に・どんな課題に向けて書くか）
- `context/marketing/content_log.md` — 過去の記事・LP・投稿と各反応（伸びた/伸びなかった傾向の学習元）
- `context/company/competitors.md` — 競合・狙うキーワード（差別化と検索接点の整理）

> 補足：content_log は「フロー（出した施策のログ）」と「ストック（今効いている勝ち筋・避ける型）」を分ける。ログは追記、勝ち筋の要約は上書きで最新に保つ。

## 2. 最初に渡すと効く自社資料（context/company/sources/ に置く）

- ブランド・トーン＆マナーガイド — 生成物の声・表記・NG表現を揃える基準になる
- 過去の記事・LP・SNS投稿（と反応数） — 自社らしさと勝ち筋をAIが学習する素材になる
- ターゲット・ペルソナ定義 — 「誰に向けて書くか」が定まり、刺さる切り口が出る
- 競合・キーワードリスト — 差別化ポイントと検索接点を踏まえた企画ができる
- 既存のクリエイティブ・図版のトーン見本 — ビジュアル指示の方向性が揃う

## 3. 鉄板スキル（universal ＋ この役割の初回セット）

**universal（全員の土台・まず作る2つ）**
1. `/daily-schedule` — 朝、カレンダーとタスクから15分刻みの工程表を作る。企画・制作・分析をブロックで組む
2. `/mtg-prep` — 編集会議・施策レビュー前に、論点・前回決定・関連データを整理（「共有用」と「社内メモ」の2部構成）

**マーケの鉄板（universal に足す）**
3. `/clip` — 競合動向・トレンド・ネタ元の記事を要約・タグ付けして溜める（企画の原料置き場）
4. `/company-research` — ターゲット業界・競合を深掘りした調査カルテを生成する（企画とポジショニングの土台）

**次に足す（優先順）**: コンテンツ下書き系（記事・LP・SNSを自社トンマナで起こす）→ SNS運用系（投稿カレンダー・各媒体の下書き）

## 4. マーケの Agent Team テンプレ — コンテンツ制作チーム

質の高い記事・LPは、単発より**分担チーム**で網羅性と説得力が上がる。`.claude/agents/` に定義し、スキルから起動する。

| エージェント | 役割 |
| --- | --- |
| リサーチ | テーマの事実・統計・事例・競合発信を収集 |
| 構成設計 | 読者の課題に沿って見出し・流れを設計 |
| ライター | 自社トンマナで本文を執筆 |
| SEO・校正 | キーワード最適化・表記統一・誤字脱字の点検 |
| レビュアー | 読者視点で刺さるか・誇張がないかを指摘 |

> フロー：リサーチ→構成設計→執筆→SEO/校正→レビュー。初回は単発のネタ収集・競合リサーチから。慣れたらこのチームを組む（本体 第2部 Step 8 参照）。

## 5. この役割ならではの注意（Gotcha）

- **トンマナを外さない**：生成物は `context/company/brand.md` を必ず参照させてから書かせる。ブランドの声・指定表記・NGワード（言い回し・誇大表現・禁止語）を外すと、量産するほど世界観が崩れる。媒体ごとの文字数・口調の違いも先に渡す。出す前にブランド表記とNGワードだけを最終チェックする。
- **数字・効果を盛らない**：「導入で◯◯％改善」「業界No.1」等は、裏取り・根拠のないまま載せない。実績・比較・優位性は客観事実のみにし、競合の社名つき否定評価はしない。外部公開物は `.claude/rules/external-materials.md` 相当のチェックを通す（薬機法・景表法に触れる表現は特に注意）。
